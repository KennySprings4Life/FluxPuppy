Operations

Visibility Name Description
public LineGraph(id: GraphView, title: String, xAxis: String, yAxis: String, color: int)

Constructor for a live line graph. Allows the graph to have points added to it dynamically. To add points to the graph, use the addPoints method.
public LineGraph(id: GraphView, title: String, xAxis: String, yAxis: String, color: int, fileContent: String)

Constructor for a static line graph. Does not allow points to be added to it dynamically. Instead, takes in a file string, specifying points to add to the graph. The file string should be formatted in the following way: <x-value>,<y-value> <x-value>,<y-value> <x-value>,<y-value> ... and so on.
public addPoint(value: float, time: float): void

Adds a point to a dynamic graph at the specified time and value. If the graph has been initialized as a static graph, then the method does not add the point.
public reset(): void

Resets the graph. This simply removes all previously saved data points, and resets the points counter.
public enableZoom(): void

On the graph screen, this method is called when the "Enable Zoom" button is pressed. Simply tells the graph that it is now in zoomable mode, and adjust the required settings to make it zoomable.
public disableZoom(): void

On the graph screen, this method is called when the "Disable Zoom" button is pressed. Simply tells the graph that it is now not in zoomable mode, and adjust the required settings to no longer make it zoomable.
public addRegLine(yIntercept: double, slope: double): void

Adds a regression line to the graph using a y intercept and a slope. Currently uses the same color as the points on the graph.
public resetZoom(): void
