Operations

Visibility Name Description
public GraphManager(activity: Activity, graphIds: GraphView, textIds: TextView)

Constructor for the GraphManager. Finds the start time, initializes the graphs, and sets other class member variables.
public run(): void

Implementation of the runnable interface. Reads in data from the instrument, calculates the time that data came in, and adds the data to the graphs.
public updateData(data: String): void

This method is used by the SerialReader object to update the most recent full line from the gas analyzer.
public deconstruct(): void

Runs when the "Finalize" button is clicked. Tells the graph update loop to stop.
public startlogging(duration: int): void

Runs when the "Start Log" button is clicked. Initializes the data log, and toggles the variable which specifies if we should be saving points.
public stoplogging(): void

Runs when the "Stop Log" button is clicked. Toggles the variable which specifies if we should be saving points.
public resetGraphs(): void

When the "Start Logging" button is pressed, the graph should be reset. This method calls each graphs individual reset method.
public enableZoom(): void

When the "Enable Zoom" button is pressed, each graph's individual enableZoom method should be called.
public disableZoom(): void

When the "Disable Zoom" button is pressed, each graph's individual disableZoom method should be called.
public isEmpty(): boolean

Not linked to any button, but is used to test if the "Finalize" button should be enabled. If the graph manager is "empty", meaning if it has recorded no data, then the finalize button should not be clickable.
public toString(): String

Allows the graph manager to be printed. This is used for storing the data of all the graphs in a CSV file.
private getData(): String

Simple getter function. Returns the last full line received from the serial reader.
private addPoints(data: String, time: long): void

Takes in a string of data from the instrument, converts it into a DataSeries, adds that DataSeries to the log, and updates each graph using the new information.
