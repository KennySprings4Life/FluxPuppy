Description

Constructor for a static line graph. Does not allow points to be added to it dynamically. Instead, takes in a file string, specifying points to add to the graph. The file string should be formatted in the following way: <x-value>,<y-value> <x-value>,<y-value> <x-value>,<y-value> ... and so on.
