Class used to communicate with the USB serial connection. More or less, it is just a really glorified string builder.

Constructor for the serial reader. Initializes all the class member variables.
public addChar(input: String): void

This method is called every time the USB port picks up a piece of data from the instrument. Makes sure it is not empty or a space, then adds it to the string being built. If the message contains a line break, then we know it is the end of the current message.
